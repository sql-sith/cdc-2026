# Network Intro, VMs, and Linux Installation, week 1

## Meeting notes

* **Virtual Machines (VMs)**
* How to create and configure VMs for CDC use.
* Tips on choosing hypervisors (likely VirtualBox or VMware).
* Snapshot management and resource allocation strategies.
* **Linux Installation**

  * Step-by-step guide to installing a Linux distro (likely Ubuntu or Debian).
  * Partitioning, user setup, and post-install tasks.
  * Introduction to terminal basics and package management (`apt`, `yum`, etc.).
* **Networking Basics**

  * IP addressing, subnetting, and routing principles.
  * How these concepts apply to CDC competition environments.
  * Tools like `ping`, `traceroute`, and `nmap` for diagnostics.

## CDC-Specific Guidance

* Emphasis on reproducibility : Keep clean VM templates and document your setup.
* Practice Linux commands : Especially those related to user management and service control.
* Understand your network : CDC scenarios often involve segmentation and isolation.

## Follow-up tasks

* Set up at least one Linux VM and practice installing from scratch.
* Learn how to configure static IPs and troubleshoot connectivity.
* Explore basic firewall rules and service hardening techniques.
