# 🧭 OWASP Juice Shop — Student Resource Sheet

Welcome to OWASP Juice Shop!
This guide gives you the  **right kind of help** : resources that teach you how to think like an application security tester without giving away all the answers.

Use these as you explore, experiment, and break things safely.

## 📘 1. Built‑In Learning Tools (Start Here)

Juice Shop includes its own progressive hint system.

### **Score Board → Lightbulb Icon → Hint**

* Hints are intentionally minimal.
* They nudge you toward the right idea without spoiling the solution.
* Use these before searching online.

## 🎓 2. OWASP Juice Shop Companion Guide (Official)

A conceptual guide that explains:

* How web vulnerabilities work
* Why Juice Shop behaves the way it does
* How to think like an attacker *and* a defender

This is **not** a walkthrough.
It’s perfect for learning the fundamentals behind each challenge.

## 🧪 3. PortSwigger Web Security Academy (Highly Recommended)

These free labs teach the exact skills you need for Juice Shop:

* SQL injection
* XSS
* Broken authentication
* Access control flaws
* API manipulation
* Business logic attacks

Each lab includes:

* A short explanation
* A hands‑on exercise
* A solution you can reveal *after* trying

If you want to build real skills, this is the best place to practice.

## 🧭 4. ZAP (Zed Attack Proxy) for Manual Testing

You’ll use ZAP to:

* Intercept and modify HTTP requests
* Replay and experiment with parameters
* Explore how the app behaves under unexpected input

This is how real security testers work.
Automation is useful, but  **manual exploration teaches you the craft** .

## 📺 5. YouTube Channels for Technique (Not Solutions)

These creators focus on  *how to think* , not just “click here to win.”

### **John Hammond**

* Clear explanations
* Great for beginners
* Focuses on methodology

### **HackerSploit**

* Practical demonstrations
* Good for understanding tools like ZAP

These are helpful when you want to understand *why* something works.

## 🧩 6. Juice Shop CTF Mode (Optional)

If you prefer a structured, gamified experience:

* CTF mode breaks challenges into clear objectives
* You solve them one at a time
* Great for self‑paced learning

Ask your instructor if this mode is enabled.

## 🛠️ 7. What You Should Practice

To get the most out of Juice Shop, try to:

* Intercept requests and modify them
* Change parameters and observe behavior
* Look for hidden API endpoints
* Explore how authentication works
* Try unexpected inputs
* Think about how you would fix the vulnerability

This is where the real learning happens.

## 🧠 8. What *Not* to Do (At Least at First)

Avoid:

* Full walkthroughs
* Automated scanners
* Copy‑paste solutions

You'll learn far more by experimenting and reasoning through the problem.

## 🏁 Final Advice

Juice Shop is designed to be broken — but it’s also designed to teach you how to think.
Use the resources above to build your skills, not just to get answers.

If you get stuck:

* Revisit the hints
* Try a different angle
* Use ZAP to inspect what’s really happening
* Ask good questions

You'll be surprised how quickly things start to click.
